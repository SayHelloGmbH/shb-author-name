Provides a block which shows the name of the current author (on the author archive page or on the single post view).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/shb-archivetitle-author` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. You can now add the “Author name” block.
